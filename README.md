# 🐍 7-Day Python Preparation Sprint for Generative AI Engineer Interviews

> Strengthen interview-ready Python skills for a Generative AI Engineer role by building practical coding fluency in **data handling**, **model utilities**, **training workflows**, **debugging**, and **deployment-oriented scripting** relevant to LLM, GAN, VAE, and MLOps use cases.

├── day2/          # NumPy, Pandas & Data Preprocessing

├── day5/          # Debugging, Profiling & Optimization

---

## 📅 Day-Wise Task Breakdown

| Day | Focus Area | Contents |
|-----|-----------|----------|
| **Day 1** | Python Fundamentals & Data Handling | List/dict comprehensions, file I/O, data loading patterns used in ML pipelines |
| **Day 2** | NumPy, Pandas & Preprocessing | Array ops, DataFrame manipulations, normalization, train/test splits |
| **Day 3** | Model Utilities & Helper Functions | Loss functions, metric calculators, checkpoint savers, config parsers |
| **Day 4** | Training Workflows | Training loops, batch generators, learning rate schedulers, logging utilities |
| **Day 5** | Debugging, Profiling & Optimization | `cProfile`, `timeit`, memory profiling, common ML bug patterns |
| **Day 6** | Deployment Scripting | FastAPI/Flask stubs, serialization, environment configs, CLI tools |
| **Day 7** | End-to-End Mini Project | Intent Router app: n modern GenAI systems, instead of sending every query to an expensive LLM, we use a fast, lightweight model to classify the user's intent and route it to the correct specialized agent.

## 📋 Deliverables

### ✅ Deliverable 1 — Day-wise Scripts / Notebooks
A GitHub-ready Python practice folder with 7 day-wise scripts or Jupyter notebooks covering the topics above.

### ✅ Deliverable 2 — Interview Notes Document
A concise reference doc (`notes/`) covering:
- Python patterns (generators, decorators, context managers)
- Optimization techniques
- Debugging strategies
- ML workflow utilities

### ✅ Deliverable 3 — Mini End-to-End Project (`project/`)
A self-contained Python project that:
- Preprocesses a dataset
- Trains a simple model
- Evaluates outputs
- Exposes reusable utility functions

### ✅ Deliverable 4 — Proof Pack (`proof_pack/`)
- Code screenshots
- Sample outputs and logs
- Timing comparisons
- Short self-review summary
